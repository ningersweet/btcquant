## 1. 技术指标特征 (24个)

### 1.1 EMA - 指数移动平均 (3个)

**特征名称**: `ema_7`, `ema_25`, `ema_99`

**计算公式**:
```python
ema = close.ewm(span=period, adjust=False).mean()
feature = (ema - close) / close
```

**含义**:
- 衡量价格相对于移动平均线的偏离程度
- 正值：价格低于均线（可能超卖）
- 负值：价格高于均线（可能超买）

**参数**:
- period = [7, 25, 99]
- 短期、中期、长期趋势

**取值范围**: 通常在 [-0.1, 0.1]

**问题**:
- ✅ 归一化处理正确
- ⚠️ 可能需要添加更多周期

**优化建议**:
```python
# 添加 EMA 交叉特征
ema_7_25_cross = (ema_7 - ema_25) / close
ema_25_99_cross = (ema_25 - ema_99) / close
```

### 1.2 RSI - 相对强弱指数 (1个)

**特征名称**: `rsi_14`

**计算公式**:
```python
delta = close.diff()
gain = delta.where(delta > 0, 0)
loss = (-delta).where(delta < 0, 0)
avg_gain = gain.ewm(span=14, adjust=False).mean()
avg_loss = loss.ewm(span=14, adjust=False).mean()
rs = avg_gain / avg_loss
rsi = 100 - (100 / (1 + rs))
feature = rsi / 100  # 归一化到 [0, 1]
```

**含义**:
- 衡量价格变动的速度和幅度
- < 0.3: 超卖区域
- > 0.7: 超买区域
- 0.5: 中性

**参数**:
- period = 14（标准参数）

**取值范围**: [0, 1]

**问题**:
- ✅ 归一化处理正确
- ⚠️ 只有一个周期

**优化建议**:
```python
# 添加多周期 RSI
rsi_periods = [6, 14, 24]
# 添加 RSI 背离特征
rsi_divergence = detect_divergence(price, rsi)
```

### 1.3 MACD - 移动平均收敛发散 (3个)

**特征名称**: `macd`, `macd_signal`, `macd_hist`

**计算公式**:
```python
ema_fast = close.ewm(span=12, adjust=False).mean()
ema_slow = close.ewm(span=26, adjust=False).mean()
macd_line = ema_fast - ema_slow
signal_line = macd_line.ewm(span=9, adjust=False).mean()
histogram = macd_line - signal_line

# 归一化
macd = macd_line / close
macd_signal = signal_line / close
macd_hist = histogram / close
```

**含义**:
- **macd**: 快慢均线差值，趋势方向
- **macd_signal**: MACD 的平滑线
- **macd_hist**: MACD 与信号线的差值，动量强度

**参数**:
- fast = 12, slow = 26, signal = 9（标准参数）

**取值范围**: 通常在 [-0.05, 0.05]

**问题**:
- ✅ 归一化处理正确
- ✅ 包含完整的 MACD 组件

**优化建议**:
```python
# 添加 MACD 交叉信号
macd_cross = np.sign(macd - macd_signal)
# 添加 MACD 背离
macd_divergence = detect_divergence(price, macd)
```

### 1.4 ATR - 平均真实波幅 (1个)

**特征名称**: `atr_14`

**计算公式**:
```python
high_low = high - low
high_close = abs(high - close.shift(1))
low_close = abs(low - close.shift(1))
true_range = max(high_low, high_close, low_close)
atr = true_range.ewm(span=14, adjust=False).mean()
feature = atr / close  # 归一化
```

**含义**:
- 衡量市场波动率
- 值越大，波动越剧烈
- 用于设置止损止盈

**参数**:
- period = 14（标准参数）

**取值范围**: 通常在 [0.01, 0.1]

**问题**:
- ✅ 归一化处理正确
- ⚠️ 只有一个周期

**优化建议**:
```python
# 添加 ATR 百分位
atr_percentile = atr.rolling(100).rank(pct=True)
# 添加波动率变化
atr_change = atr.pct_change()
```

### 1.5 Bollinger Bands - 布林带 (5个)

**特征名称**: `bb_upper`, `bb_middle`, `bb_lower`, `bb_width`, `bb_pct`

**计算公式**:
```python
middle = close.rolling(window=20).mean()
std = close.rolling(window=20).std()
upper = middle + (std * 2)
lower = middle - (std * 2)

# 归一化
bb_upper = (upper - close) / close
bb_middle = (middle - close) / close
bb_lower = (lower - close) / close
bb_width = (upper - lower) / close
bb_pct = (close - lower) / (upper - lower)  # %B
```

**含义**:
- **bb_upper/lower**: 价格到上下轨的距离
- **bb_middle**: 价格到中轨的距离
- **bb_width**: 布林带宽度，波动率指标
- **bb_pct**: 价格在布林带中的位置
  - < 0: 价格低于下轨（超卖）
  - > 1: 价格高于上轨（超买）
  - 0.5: 价格在中轨

**参数**:
- period = 20, std = 2（标准参数）

**取值范围**:
- bb_upper/middle/lower: [-0.1, 0.1]
- bb_width: [0.01, 0.2]
- bb_pct: [0, 1]（可能超出）

**问题**:
- ✅ 归一化处理正确
- ✅ 特征完整

**优化建议**:
```python
# 添加布林带挤压
bb_squeeze = bb_width < bb_width.rolling(100).quantile(0.2)
# 添加布林带突破
bb_breakout = (bb_pct > 1) | (bb_pct < 0)
```

### 1.6 OBV - 能量潮 (2个)

**特征名称**: `obv_norm`, `obv_ema_norm`

**计算公式**:
```python
direction = np.sign(close.diff())
obv = (direction * volume).cumsum()
obv_ema = obv.ewm(span=20, adjust=False).mean()

# 归一化到 [0, 1]
obv_max = obv.rolling(window=100).max()
obv_min = obv.rolling(window=100).min()
obv_range = obv_max - obv_min

obv_norm = (obv - obv_min) / obv_range
obv_ema_norm = (obv_ema - obv_min) / obv_range
```

**含义**:
- 衡量成交量的累积流向
- 上涨日成交量为正，下跌日为负
- 用于确认价格趋势

**参数**:
- ema_period = 20
- rolling_window = 100

**取值范围**: [0, 1]

**问题**:
- ✅ 归一化处理正确
- ⚠️ 使用 100 周期滚动窗口可能不够稳定

**优化建议**:
```python
# 使用更稳定的归一化
obv_zscore = (obv - obv.rolling(100).mean()) / obv.rolling(100).std()
# 添加 OBV 背离
obv_divergence = detect_divergence(price, obv)
```

## 2. 结构特征 (15个)

### 2.1 High/Low - 高低点 (6个)

**特征名称**: `highest_high_20`, `lowest_low_20`, `highest_high_50`, `lowest_low_50`, `highest_high_100`, `lowest_low_100`

**计算公式**:
```python
for period in [20, 50, 100]:
    highest = high.rolling(window=period).max()
    lowest = low.rolling(window=period).min()
    
    highest_high = (highest - close) / close
    lowest_low = (close - lowest) / close
```

**含义**:
- **highest_high**: 价格距离周期内最高点的距离
- **lowest_low**: 价格距离周期内最低点的距离
- 用于判断支撑阻力位

**参数**:
- periods = [20, 50, 100]

**取值范围**: [0, 0.5]（通常）

**问题**:
- ✅ 归一化处理正确
- ✅ 多周期覆盖

**优化建议**:
```python
# 添加突破信号
breakout_high = close > highest.shift(1)
breakout_low = close < lowest.shift(1)
```

### 2.2 Round Number - 整数关口 (3个)

**特征名称**: `round_dist_pct`, `round_above_pct`, `round_below_pct`

**计算公式**:
```python
base = 1000  # 整数关口基数
round_above = np.ceil(close / base) * base
round_below = np.floor(close / base) * base

dist_above = round_above - close
dist_below = close - round_below
nearest_dist = np.minimum(dist_above, dist_below)

round_dist_pct = nearest_dist / close
round_above_pct = dist_above / close
round_below_pct = dist_below / close
```

**含义**:
- 价格距离整数关口的距离
- 整数关口通常是心理支撑/阻力位
- 例如：$60,000, $65,000, $70,000

**参数**:
- base = 1000

**取值范围**: [0, 0.02]（通常）

**问题**:
- ✅ 概念正确
- ⚠️ base=1000 可能不够灵活

**优化建议**:
```python
# 动态调整 base
if close < 10000:
    base = 100
elif close < 50000:
    base = 500
else:
    base = 1000
```

### 2.3 Time - 时间特征 (5个)

**特征名称**: `hour`, `day_of_week`, `is_weekend`, `hour_sin`, `hour_cos`

**计算公式**:
```python
dt = pd.to_datetime(timestamp, unit='ms')

hour = dt.dt.hour / 24  # 归一化到 [0, 1]
day_of_week = dt.dt.dayofweek / 7  # 0=周一, 6=周日
is_weekend = (dt.dt.dayofweek >= 5).astype(int)

# 周期性编码
hour_sin = np.sin(2 * np.pi * dt.dt.hour / 24)
hour_cos = np.cos(2 * np.pi * dt.dt.hour / 24)
```

**含义**:
- 捕捉时间周期性模式
- 日内交易时段特征
- 周末效应

**参数**: 无

**取值范围**:
- hour, day_of_week: [0, 1]
- is_weekend: {0, 1}
- hour_sin, hour_cos: [-1, 1]

**问题**:
- ✅ 周期性编码正确
- ⚠️ 缺少月份、季度特征

**优化建议**:
```python
# 添加更多时间特征
```

### 3.3 特征质量问题

#### 问题1: NaN 处理不当
```python
# 当前做法
result_df = result_df.fillna(0)
```

**影响**:
- 丢失"缺失"信息
- 引入错误的 0 值
- 影响模型学习

**解决方案**:
```python
# 方案1: 前向填充
result_df = result_df.fillna(method='ffill')

# 方案2: 使用 LightGBM 原生 NaN 支持
# 不填充，让模型自己处理

# 方案3: 智能填充
for col in result_df.columns:
    if 'pct' in col or 'ratio' in col:
        result_df[col].fillna(0)  # 百分比类填 0
    elif 'price' in col:
        result_df[col].fillna(method='ffill')  # 价格类前向填充
    else:
        result_df[col].fillna(result_df[col].median())  # 其他用中位数
```

#### 问题2: 特征归一化不一致

**当前状态**:
- 有些特征除以 close 归一化
- 有些特征使用 min-max 归一化
- 有些特征使用百分比

**影响**:
- 特征尺度不一致
- 影响模型训练效率

**解决方案**:
```python
# 统一使用 StandardScaler
from sklearn.preprocessing import StandardScaler

scaler = StandardScaler()
features_scaled = scaler.fit_transform(features)
```

#### 问题3: 特征相关性高

**问题**: 许多特征高度相关
- ema_7, ema_25, ema_99 相关性高
- bb_upper, bb_middle, bb_lower 相关性高

**影响**:
- 特征冗余
- 增加模型复杂度
- 可能过拟合

**解决方案**:
```python
# 特征选择
from sklearn.feature_selection import SelectKBest, f_regression

selector = SelectKBest(f_regression, k=20)
X_selected = selector.fit_transform(X, y)

# 或使用 PCA 降维
from sklearn.decomposition import PCA
pca = PCA(n_components=0.95)  # 保留 95% 方差
X_pca = pca.fit_transform(X)
```
